## 🎯 How to Use

### Using Entry Templates
1. Click the "✍️" button to create a new entry
2. Click the "📋 Templates" button in the composer header
3. Browse templates by category or view all
4. Click any template to auto-fill the structure
5. Customize the content to fit your needs
6. Save your structured entry

### Adding Tags & Categories
1. Click the "📝" button to create a new entry
2. Select your mood as usual
3. **Choose Category**: Pick from the category buttons
4. **Add Tags**: Type in the tags field or select from suggestions
5. Write your entry and save

### Exporting Your Diary
1. Click the "📤" button in the header
2. Choose your export format (JSON, Markdown, or Text)
3. Select date range and optional tag filters
4. Click "Export" to download

### Customizing Your Experience
1. Click the "⚙️" button in the header
2. Experiment with themes, fonts, and sizes
3. Use the preview to see changes
4. Click "Save Changes" to apply

### Using Admin Panel (Hidden from Users)
1. Login with admin email (dp7800549@gmail.com)
2. **Admin Interface**: You'll automatically see the admin feed (different from normal users)
3. **Crown Button**: Click the "👑" crown button in the header (only visible to admin)
4. **Statistics View**: See overall app analytics
5. **Entries View**: Browse all user entries
6. **Search & Filter**: Find specific entries or users
7. **Content Management**: Delete inappropriate content
8. **Real-time Monitoring**: Live activity updates

**Important**: Normal users will never see any admin features, buttons, or know the admin panel exists.
